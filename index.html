使用七牛云作为图床保存图片

发表于 2019-01-30 | 分类于 日常

首先需要注册下七牛云, 注册完后,进入主页选择对象存储

然后新建存储空间, 填的第一项是bucket(这一项在后面配置七牛云插件的时候还需要用到), 存储区域就近选择就好, 访问控制选择公开空间, 最后确认创建

我使用的是hexo的插件的hexo-qiniu-sync(这是链接)

首先在hexo主目录下进行安装:

安装完后, 将配置信息写入hexo主目录下的_config.yml中

接下来讲解下需要更改的配置:
首先是bucket, 这个在之前讲过,就是新建存储空间时的bucket

然后是access_key和secret_key, 这两个参数在个人中心的密钥管理下可以找到, secret_file可以注释掉, 不用管

之后是dirPrefix以及urlPrefix, urlPrefix在对象存储页的内容管理中的外部默认域名项可以找到, 如果设置了dirPrefix, 则需要把这个目录名加到urlPrefix, 例如aaaaaaaa.bbbbb.gdipper.com/static, 后面加了个static

最后是up_host, 这个由之前创建空间时, 选择的区域而定, 在文档中心可以找到

至于local_dir, 是存放图片及其他资源的目录, 在执行hexo的时候会自动生产

flask微电影项目(一)

发表于 2019-01-22 | 分类于 后端

放寒假了，在家里无聊准备随便搞点事情，然后就在b站上找视频，找到了中国flask微电影项目，刚好练练手

前言

win10下, 选择python3 + mysql 5.7.25 + pycharm

安装虚拟环境

首先需要安装virtualenv, 一句命令就可以搞定, 但安装的时候要选择国内源, 我用的是豆瓣

配置虚拟环境

新建一个项目目录后, 进入并执行virtualenv venv, 之后会出现一个venv目录,现在我们进入虚拟环境, cd到venv/Scripts/, 执行./activate

重点来了,我的power shell突然报错

无法加载文件 C:\Users\wfengcd\Desktop\movie\venv\Scripts\activate.ps1，因为在此系统上禁止运行脚本。

问题倒是很容易解决, 以管理员的身份启动power shell,执行set-executionpolicy remotesigned然后选择Y, 这样就有权限执行脚本了


执行完./activate就进入了虚拟环境, 在命令行前面, 会出现一个(venv), 想要退出虚拟环境, 执行deactivate即可

falsk安装

安装前可以执行pip freeze检测下环境, 一般情况下, 检测出来应该是空

然后安装flask, 用的还是豆瓣源

很快就安装完了, 此时再执行pip freeze就会发现多了不少东西

自此,大功告成
